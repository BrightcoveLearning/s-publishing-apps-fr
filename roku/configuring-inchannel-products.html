Présentation

Les produits in-channel sont des solutions de monétisation que vous pouvez proposer sur vos canaux, comme un abonnement mensuel ou annuel. Ce document explique comment créer des produits in-channel pour monétiser vos canaux d'application Roku.

Si des chaînes bêta sont créées à des fins de test, elles expirent 45 jours après leur création.

Création d'un produit in-channel dans Roku

Découvrez comment ajouter un produit in-channel sur la page Développeurs Roku en suivant ces étapes:

  1. Dans la barre de navigation supérieure, cliquez sur l'icône de profil.

  2. Dans le menu déroulant, cliquez sur l'option Développeurs.

  3. Dans la barre de navigation supérieure, sélectionnez le lien Tableau de bord.

  4. Sélectionnez Gérer mes produits in-channel.

  5. Cliquez sur le bouton Ajouter un nouveau produit.
  6. Dans le champ Canaux, sélectionnez les chaînes où le produit sera disponible à la vente.
  7. Remplissez le formulaire (voir la capture d'écran ci-dessous la liste des étapes):
    1. Pour la catégorie de produits, sélectionnez Vidéo.
    2. Dans le champ Nom du produit, tapez le nom qui sera visible pour les clients. Vous pouvez ajouter le nom du produit dans différentes langues en cliquant sur le bouton Ajouter un nom de produit dans une autre langue.
    3. Pour l'identifiant du produit, saisissez votre ID de référence interne.
      Notez que l'ID de référence interne doit être unique, et vous en aurez besoin ultérieurement pour ajouter votre produit In-Channel à la configuration du CMS Brightcove {{site.product_short}}.

    4. Pour le type d'achat, sélectionnez le type d'abonnement en fonction de la façon dont vos clients sont facturés pour votre produit in-channel. Dans cet exemple, un abonnement mensuel.

      Notez que l'intervalle d'abonnement pour Roku est différent de celui des autres boutiques. Pour plus de détails, consultez la section Prise en charge des intervalles d'abonnement du document Utilisation d'un plan SVOD
    5. Pour l'option Niveau de prix, sélectionnez un niveau de prix basé sur le prix de votre produit in-channel.
  8. Dans la section Essais et offres, en fonction de votre stratégie produit in-channel, vous pouvez sélectionner Essai gratuit et saisir un numéro de durée dans la Durée de l'essai et sélectionnez Jours ou Mois à partir de l'heure de saisie.
  9. Allez au bas de la page, cochez la case Compensé à la vente et cliquez sur le bouton Enregistrer.
  10. Après la création, assurez-vous que votre produit est visible sur la page Gérer mes produits in-channel.
    Vous pouvez créer un nombre illimité de produits in-channel supplémentaires en fonction du type d'abonnement que vous souhaitez proposer. Exemple: annuel ou mensuel.

Associez votre produit in-channel à une chaîne dans Brightcove {{site.product_short}}

Une fois vos produits in-channel créés, accédez à la plateforme Brightcove {{site.product_short}} pour associer vos nouveaux produits in-channel à une chaîne.

  1. Dans le menu supérieur, cliquez sur l'onglet Commerce.
  2. Dans le menu de gauche, sélectionnez l'option Packages.
  3. Cliquez sur votre forfait.
  4. Dans l'onglet Forfait, descendez à la section Détails des prix.
  5. Dans le champ Prix, saisissez le prix de votre produit in-channel.
  6. Dans le champ Devise, sélectionnez le type de devise.
  7. Dans l'ID de magasin Roku, tapez l'identifiant de votre produit in-channel créé dans la plateforme Roku Developers.
  8. Allez au bas de la page et cliquez sur le bouton Modifier le package pour enregistrer vos modifications.
