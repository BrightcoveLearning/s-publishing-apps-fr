Présentation

Un utilisateur doit être ajouté à votre compte Roku pour Brightcove. Contactez l'équipe de livraison OTT de Brightcove pour obtenir une adresse e-mail à utiliser.

Instructions

Effectuez les étapes suivantes pour ajouter une adresse utilisateur à votre compte Roku:

  1. Connectez-vous à votre compte développeur Roku.
  2. Cliquez sur l'icône de l'utilisateur, puis sur Mon compte.
  3. Cliquez sur Gérer l'accès des utilisateurs.
  4. Cliquez sur Ajouter un utilisateur.
  5. Remplissez le formulaire:
    • Saisissez l'adresse e-mail spécifiée par l'équipe de livraison OTT de Brightcove.
    • Ajoutez le rôle d'administrateur.
    • Entrez dans Brightcove pour l'organisation.
  6. Cliquez sur Créer.
