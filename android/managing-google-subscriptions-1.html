Présentation

Ce document explique comment utiliser la console Google Play pour créer un compte de service, puis effectuer quelques tâches de configuration.

Créer un compte de service

Effectuez les étapes suivantes pour créer un compte de service:

  1. Accédez à la console Google Play.
  2. Dans le coin supérieur droit, cliquez sur Go to Play Console.
  3. Dans le menu de navigation de gauche, sélectionnez Paramètres â†' Compte développeur â†' Accès à l'API.
  4. Dans la boîte de dialogue qui apparaît, cliquez sur Google Cloud Platform.
  5. Sur la page vers laquelle vous êtes redirigé, cliquez sur + CRÉER UN COMPTE DE SERVICE.
  6. Dans la boîte de dialogue qui s'ouvre, pour le nom du compte de service, saisissez Abonnements Android, puis cliquez sur CRÉER.
  7. À l'étape 2, dans la liste déroulante, choisissez le rôle à attribuer au propriétaire, puis cliquez sur CONTINUER.
  8. À l'étape 3, cliquez sur TERMINÉ.

Configuration du compte de service

Effectuez les étapes suivantes pour configurer votre compte de service:

  1. Pour le compte de service qui vient d'être créé, sous Actions, cliquez sur les trois points verticaux. Cliquez ensuite sur Gérer les clés.
  2. Dans les informations Clés, sélectionnez AJOUTER UNE CLÉ â†' Créer une nouvelle clé
  3. Dans les informations de clé privée, laissez le type de clé sur JSON, puis cliquez sur CREATE.
  4. Assurez-vous que le message suivant s'affiche:
  5. Vous devez envoyer le fichier de clé privée par e-mail à votre gestionnaire d'intégration.
  6. Vous pouvez fermer l'onglet que vous avez ouvert à l'étape 5 et cliquer sur Terminé dans la boîte de dialogue qui a lancé le processus de création de votre compte de service.
  7. Dans la section Comptes de service, cliquez sur Accorder l'accès en regard du compte de service que vous venez de créer.
  8. Sur la page Inviter un utilisateur, dans la section Autorisations du compte, cochez la case Admin (toutes les autorisations).
  9. En bas à droite de la page, cliquez sur Inviter un utilisateur.
  10. Dans la boîte de dialogue qui s'ouvre, cliquez sur Envoyer une invitation.
