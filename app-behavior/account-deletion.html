Conformité du consentement

Apple exige que les applications iOS/tvOS disposent d'un bouton Supprimer pour supprimer le compte d'un spectateur. Consultez le document Guide de révision de l'App Store d'Apple, section Légal - Collecte et stockage des données pour plus de détails.

Mise en œuvre de Brightcove {{site.product_short}}

Dans les options Paramètres -> Mon compte de l'application, une option Supprimer le compte est présente

Lorsque l'utilisateur clique sur le bouton, {{site.product_short}} effectue les tâches suivantes:

  • Confirme que le spectateur souhaite supprimer le compte.
  • Alerte le spectateur qu'il doit annuler tous les abonnements actifs auprès des boutiques d'autres plateformes avant que le compte {{site.product_short}} puisse être supprimé.
  • Le backend de {{site.product_short}} vérifie tous les abonnements actifs du spectateur auprès des boutiques d'autres plateformes qu'il n'y a pas d'abonnement actif.
  • Suppression des informations sur les spectateurs de Stripe et Evergent (nom, coordonnées, données de paiement)
  • Le service de gestion des utilisateurs de {{site.product_short}} supprime le compte et toutes les données associées à ce compte, telles que les préférences, les favoris, les vidéos récemment regardées, etc.
  • Informe l'utilisateur que le compte est supprimé, via un message dans une alerte d'application, lorsque toutes les conditions sont remplies et que les processus de suppression sont terminés.

Notez que si un utilisateur utilise un identifiant OIDC et se connecte après la suppression du compte, un compte avec ces informations d'identification sera créé, mais il s'agit d'un nouveau compte distinct et aucune donnée précédente associée à ce compte n'est récupérée.
