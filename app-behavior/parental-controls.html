Définition de la valeur

Pour définir la restriction du contrôle parental, procédez comme suit:

  1. Connectez-vous à l'application Beacon.
  2. Dans l'application, accédez à Paramètres -> Mon compte -> Contrôle parental
  3. Si c'est la première fois que vous effectuez cette tâche, lorsque vous cliquez sur l'un des boutons radio correspondant à un paramètre spécifique, vous êtes invité à saisir un code PIN. Si vous avez déjà effectué une sélection, vous serez invité à saisir le code PIN lorsque vous modifiez la sélection.

Notez ce qui suit:

  • La note sélectionnée signifie que les spectateurs peuvent voir le contenu jusqu'à ce niveau d'évaluation inclus.
  • La note sélectionnée s'applique à TOUS les utilisateurs enregistrés de l'application.
  • Si un spectateur essaie de regarder une vidéo restreinte, l'application l'invite à entrer le code PIN et, si saisi correctement, la vidéo peut être visionnée.
