Aperçu

Vous aurez besoin d'un partenariat Samsung Apps TV et d'un groupe de partenaires si vous envisagez de lancer votre application en dehors des États-Unis ou si vous souhaitez accéder à certaines API.

Voici le processus de demande de partenariat:

  1. Si vous n'en avez pas déjà un, créez un groupe public. Cela est nécessaire pour la demande de partenariat.

  2. Demandez un accord hors ligne avec le gestionnaire de contenu Samsung. Dans certains cas, cette exigence peut être levée par le gestionnaire de contenu.

  3. Une fois que le gestionnaire de contenu Samsung aura approuvé votre demande, vous deviendrez partenaire Samsung Apps TV.

Création d'un groupe public

Tout d'abord, si vous n'en avez pas déjà un, créez un groupe public.

  1. Dans un navigateur, accédez à la page Samsung Sign in.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

  3. Si vous n'êtes pas encore membre d'un groupe public, créez un nouveau groupe. Dans le menu de navigation de gauche, sélectionnez Gestion des groupes.

  4. Cliquez sur le bouton Aller à Créer un groupe-fr/.

  5. Entrez des valeurs pour les informations de groupe et les informations sur le vendeur Lorsque vous avez terminé, cliquez sur le bouton Créer un groupe.

Demande de partenariat

Ensuite, demandez un partenariat.

  1. Sur l'écran d'accueil du développeur, sélectionnez le bouton Demander un partenariat dans le menu de navigation de gauche ou le lien Demande de partenariat dans la barre de navigation supérieure.

  2. Saisissez les informations du partenaire.

    La page se compose de trois sections de saisie:

    • Informations sur Samsung Contents Manager
    • Informations sur le groupe
    • Informations sur le vendeur

    Selon le statut de création de votre groupe, la page du formulaire de demande sera différente.

    • Si vous avez déjà créé un groupe, les valeurs des informations du groupe et des informations sur le vendeur seront définies à partir de votre groupe. Vous pouvez modifier les informations de votre groupe ici.
    • Si vous n'avez pas créé de groupe, tous les champs seront vides.

    Les champs Informations sur le groupe et Informations sur le vendeur sont obligatoires pour l'enregistrement.

    Étant donné que Samsung traite les demandes de partenariat en fonction du groupe, le bureau du vendeur créera votre groupe en tant que groupe public avant d'examiner votre demande.

  3. Si vous ne connaissez pas votre Samsung Contents Manager, soumettez une question à l'aide des questions et réponses individuelles. Samsung fournira l'adresse e-mail de votre gestionnaire de contenu.™

    • Dans le menu de navigation supérieur, passez la souris sur Support.
    • Sélectionnez Q&R 1:1.
    • Cliquez sur le bouton Poser une question.
  4. Lorsque vous avez terminé de remplir la page d'informations sur le partenariat, cliquez sur le bouton Demander un partenariat.

Vérification du statut du partenariat

Enfin, nous allons vous montrer comment vérifier votre statut de partenariat.

  1. Une fois que le gestionnaire de contenu Samsung aura approuvé votre demande, vous deviendrez partenaire Samsung Apps TV.

  2. Vous pouvez vérifier que vous êtes un vendeur partenaire en consultant la page d'accueil du bureau des vendeurs.
