Prérequis

Pour pouvoir installer l'application Brightcove {{site.product_short}} sur votre Samsung Smart TV, vous aurez besoin des éléments suivants:

  • Téléviseur intelligent Samsung avec système d'exploitation Tizen
  • Assurez-vous que la dernière version du micrologiciel est installée sur la Smart TV Samsung

Installation de l'application Brightcove {{site.product_short}} sur les appareils Samsung Smart TV

Une fois que vous avez reçu l'e-mail Brightcove, veuillez suivre les étapes suivantes:

  1. Dans l'e-mail envoyé par Brightcove, une pièce jointe ZIP apparaîtra. Téléchargez le fichier ZIP.
  2. Double-cliquez sur le fichier ZIP et extrayez-le du fichier. Les fichiers de versements de Brightcove {{site.product_short}} se trouvent dans ce dossier.
  3. Copiez le dossier et collez-le à la racine d'une clé USB.
  4. Insérez la clé USB dans votre Samsung Smart TV. L'application Brightcove {{site.product_short}} commencera à s'installer automatiquement.
  5. Dans la Smart TV Samsung, allez dans le menu Applications et ouvrez l'application Brightcove {{site.product_short}}.
Chaque fois que vous souhaitez installer une nouvelle version de l'application Brightcove {{site.product_short}}, vous devez supprimer la version précédemment installée, sinon vous trouverez une erreur système.

Désinstallation de l'application Brightcove {{site.product_short}}

Pour désinstaller votre application de l'appareil Samsung Smart TV, procédez comme suit:

  1. Appuyez sur le bouton principal de votre manette OneRemote.
  2. À l'aide du pavé directionnel de votre télécommande, accédez à APPS et sélectionnez-les.
  3. Sélectionnez Options.
  4. Sélectionnez l'application Brightcove {{site.product_short}}.
  5. Sélectionnez Supprimer.
  6. Sélectionnez à nouveau Supprimer.
