Informations requises

Pour pouvoir installer l'application Brightcove {{site.product_short}} sur votre appareil Apple, vous aurez besoin des éléments suivants:

  • Un lien public ou un e-mail d'invitation fourni par Brightcove.
  • Application TestFlight installée sur votre appareil Apple.

Plateformes prises en charge par Apple

  • Applications iOS: iPhone, iPad ou iPod touch exécutant iOS 8 ou version ultérieure.
  • Applications tvOS: Apple TV exécutant tvOS 12 ou version ultérieure.

Voici une vue de haut niveau du processus:

  • Installez l'application TestFlight sur votre appareil.
  • Ouvrez le lien public ou l'e-mail d'invitation qui vous a été envoyé par Brightcove.
  • Utilisez le code d'installation pour l'application Brightcove {{site.product_short}} dans l'application TestFlight.
  • Lancez l'application.

Installation de l'application iOS Brightcove {{site.product_short}}

Dans cette section, vous apprendrez comment télécharger l'application iOS Brightcove {{site.product_short}} pour la tester sur votre iPhone, iPad ou iPod touch.

  1. Installez TestFlight sur l'appareil iOS que vous utiliserez pour les tests.™
  2. Ouvrez votre e-mail d'invitation ou appuyez sur le lien public sur votre appareil iOS.
  3. Vous recevrez un code d'échange dans l'e-mail d'invitation. Copiez le code.
  4. Ouvrez l'application TestFlight sur votre appareil Apple.
  5. Cliquez sur le bouton Utiliser dans le coin supérieur droit.
  6. Collez le code d'échange Brightcove.
  7. Cliquez sur le bouton Utiliser.

Installation de l'application Brightcove {{site.product_short}} tvOS

Dans cette section, vous apprendrez comment télécharger l'application Brightcove {{site.product_short}} tvOS pour la tester sur votre Apple TV.

  1. Installez TestFlight sur votre Apple TV.
  2. Ouvrez votre e-mail d'invitation sur un appareil mobile ou un ordinateur.
  3. Vous recevrez un code d'échange dans l'e-mail d'invitation. Veillez à ce que le code soit visible afin de pouvoir le saisir ultérieurement.
  4. Ouvrez l'application TestFlight sur votre Apple TV.
  5. Cliquez sur l'onglet Utiliser et saisissez le code d'échange Brightcove.
  6. L'application Brightcove {{site.product_short}} sera installée sur votre appareil à des fins de test.
