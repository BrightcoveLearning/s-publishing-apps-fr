Présentation

Avec Brightcove {{site.product_short}}, vous recevrez une application pour appareil générée qui pourra être ajoutée à la boutique Panasonic.

  • Détails de la boutique: https://developer.vieraconnect.com/
  • Délai suggéré pour l'approbation: Trois semaines

Informations requises pour Panasonic:

  • Application
    • URL de l'application et URL de mise en œuvre
  • Description
    • 1024 caractères maximum
    • 26 lignes maximum
    • Chaque ligne comprend un maximum de 85 caractères (ou 44 caractères asiatiques)
    • Veuillez insérer des sauts de ligne le cas échéant
  • Icones
    • Quatre images d'icônes sont nécessaires:
      • 160 x 112 (PNG)
      • Icône de style de bannière 256x60 (PNG)
      • 360 x 252 (PNG)
      • Icône de cercle rond de diamètre 336x336 (PNG)
  • Capture
    • Il en faut au moins un
    • Format PNG 480 x 270 po H
    • Ils sont affichés sur la page détaillée de l'application dans le Market

Pour commencer

Pour accéder aux pages destinées aux développeurs de Panasonic, procédez comme suit:

  1. Dans un navigateur, accédez à la page de connexion des développeurs Panasonic.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

  3. Lorsque vous vous connectez, vous devriez voir le panneau de commande.

  4. Lisez la note qui se trouve à droite du bouton Soumettre une nouvelle demande. Sélectionnez le lien de la page d'accueil.

  5. Consultez les exigences de l'application. Ne vous inquiétez pas, Brightcove créera l'application avec la technologie nécessaire.

Infos sur l'application

La section Informations sur l'application vous permet de saisir les informations de base de l'application.

  1. Dans le panneau de configuration, sélectionnez le bouton Soumettre une nouvelle demande.

  2. Vous devriez voir un formulaire de saisie comme celui-ci:

  3. Dans la page Informations sur l'application, entrez des valeurs pour les champs obligatoires:

    • Type de projet: dans le menu déroulant, sélectionnez la dernière version HTML disponible.
    • URL de l'application: créez deux référentiels pour héberger vos versions d'applications à partir de Brightcove: Un pour la production et un pour la mise en scène. L'URL de l'application est l'URL de production.
    • URL intermédiaire: saisissez l'URL de test pour la création de votre application.
    • Numéro de version: saisissez la version de votre application.

  4. Cliquez sur le bouton Ajouter/Supprimer. Choisissez les langues que vous souhaitez voir disponibles pour votre application. Cliquez sur le bouton Confirmer.

  5. Saisissez des valeurs pour les champs correspondant à la langue sélectionnée. Dans cet exemple, l'anglais (Amérique du Nord) est sélectionné.

    • Langue par défaut: dans le menu déroulant, choisissez la langue initiale de votre application. Si votre application prend en charge l'anglais, Panasonic vous recommande de choisir l'anglais ((Amérique du Nord) ou l'anglais (modèle britannique)).
    • Nom de l'application: saisissez le nom que vous souhaitez utiliser pour votre application.
    • Nom abrégé - Saisissez le nom abrégé que vous souhaitez utiliser pour votre application. Il peut s'agir du même nom que le nom de l'application.
    • Mots clés - Saisissez des mots clés à utiliser par la fonction de recherche Market pour trouver votre application. Séparez les mots-clés par une virgule (ex: tennis de table, sports). Maximum de 128 caractères.
    • Description: saisissez une description pour votre application. Cela sera affiché sur la page détaillée de l'application dans le Market. Cliquez dans la zone de saisie pour plus de détails.
    • Contrat de licence d'utilisateur final - Facultatif. Saisissez le contrat de licence de l'utilisateur final.

  6. Sélectionnez la langue suivante dans la liste des langues que vous avez choisies et saisissez les valeurs des champs décrits à l'étape précédente.

  7. Cliquez sur le bouton Suivant.

Informations supplémentaires

La section Détails supplémentaires vous permet d'ajouter des détails sur le support et le contenu.

  1. Après avoir sélectionné le bouton Suivant dans la section précédente, vous devriez voir un formulaire de saisie comme celui-ci:

  2. Dans la page Informations supplémentaires, entrez des valeurs pour les champs obligatoires:

    • E-mail d'assistance: indiquez l'adresse e-mail que les utilisateurs peuvent contacter lorsqu'ils rencontrent des problèmes avec votre application.
    • Évaluation parentale - Développez le menu déroulant pour voir les définitions de chaque évaluation parentale. Sélectionnez la note qui convient à votre application.
    • Classement du contenu au Brésil - Ceci est requis si vous prévoyez de sortir au Brésil. Développez le menu déroulant pour voir les définitions de chaque classement du Brésil. Sélectionnez la note appropriée.
    • Adresse e-mail du contact - Saisissez votre adresse e-mail de contact.
    • Catégorie - Développez le menu déroulant pour afficher la liste des catégories. Sélectionnez la catégorie appropriée.
    • Prix - Normalement, les applications sur Panasonic sont téléchargeables gratuitement.
    • Achat intégré: indiquez si votre application comportera des achats intégrés.
    • Market Place - Cliquez sur le bouton Sélectionner un marché pour voir la liste des pays. Choisissez Tous les pays ou sélectionnez les pays dans lesquels vous souhaitez que votre application soit disponible. Lorsque vous avez terminé, cliquez sur le bouton Terminé.
    • Instructions pour l'assurance qualité - Incluez toutes les instructions qui peuvent être utiles, y compris l'ajout d'un bouton, la modification d'un lien, etc. Si nécessaire, incluez des comptes de test pour votre application.

  3. Cliquez sur le bouton Suivant.

Télécharger des actifs

La section Upload Assets vous permet d'ajouter des icônes et des images.

  1. Après avoir sélectionné le bouton Suivant dans la section précédente, vous devriez voir un formulaire de saisie comme celui-ci:

  2. Dans la page Charger des ressources, entrez des valeurs pour les champs obligatoires:

    • Description de l'application Doc - Créez un document qui détaille le fonctionnement de votre application. Il peut s'agir du même document que celui que vous avez créé pour LG. Cliquez sur le bouton Parcourir pour localiser votre fichier. Ensuite, cliquez sur le bouton Upload.

      Notez que seuls les fichiers doc, docx ou pdf peuvent être téléchargés. La taille maximale du fichier est de 8 Mo.

    • Icône 1 - Téléchargez une icône Market, qui s'affichera sur la page de mise en vente et la page détaillée de l'application sur le Market.

      PNG 160 x 112 h

    • Icône 2 (forme circulaire) - Téléchargez une icône Firefox OS, qui sera affichée sur la page Apps Doc du Market.

      336 pixels de diamètre en forme de cercle PNG

    • Exemple d'image 1 - Il s'agit de captures d'écran de votre application, qui seront affichées sur la page détaillée de l'application dans le Market. Ajoutez jusqu'à 5 exemples d'images.

      Format PNG 480 x 270 po H

  3. Cliquez sur le bouton Suivant.

Accepter les conditions générales

La section Accepter les conditions générales vous permet de passer en revue toutes les informations que vous avez ajoutées pour votre application avant de les soumettre à la boutique. Il y a également une section contenant les conditions générales de Panasonic.

  1. Après avoir sélectionné le bouton Suivant dans la section précédente, vous devriez voir un formulaire de saisie comme celui-ci:

  2. Passez en revue les informations et les images que vous avez ajoutées à votre application.

  3. Lisez les conditions générales de Panasonic TV. Cochez la case indiquant que vous avez lu et que vous êtes d'accord.

  4. Cliquez sur le bouton Envoyer.

  5. Sur la page Liste des applications, vous devriez voir votre application enregistrée en tant que brouillon.

  6. Votre application passera au statut QA avant d'être mise en ligne.
