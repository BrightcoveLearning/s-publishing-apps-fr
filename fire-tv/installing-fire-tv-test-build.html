Introduction

This document shows you how to install a test build so you don't have to go through the app submission process before you can test your app.

To be able to install the Brightcove {{ site.product_short }} app in your Fire TV device, you will need:
  • Developer mode activated in your Fire TV device.
  • Android Debug Bridge (adb) installed on your computer.

If you want to test the App internal payment methods, such as subscriptions or event payments, please follow the Live App Testing installation instructions.

A high-level view of the process is:

  1. Enable debugging on your Amazon Fire TV device.
  2. Connect to your Fire TV device through the Android Debug Bridge (adb).
  3. Install your Brightcove {{ site.product_short }} App.
  4. Run your Brightcove {{ site.product_short }} App.

Enabling Debugging on Amazon Fire TV

To enable debugging in your Fire TV device, please follow the steps below:

  1. From the main screen of your Fire TV, select Settings.
  2. Select Device (or My Fire TV) > Developer Options.
  3. Turn on ADB Debugging.
  4. Turn on Apps from Unknown Sources.

Connecting through Android Debug Bridge (adb)

You need to use Android Debug Bridge (adb) to connect your computer to an Amazon Fire TV device for installing, testing and debugging your apps. For more information on how to enable adb please follow the Amazon and Android documentation.

Installing your Brightcove {{ site.product_short }} App

To install and test your Brightcove {{ site.product_short }} app before submitting it to the Amazon Appstore, you need to use the Android Debug Bridge (adb).

Installing your own app (outside of the Appstore) is sometimes referred to as sideloading an app.

To follow these steps, you must have already used adb to connect your development computer to your Fire TV device.
  1. Open your terminal or command line.
  2. Type the following command, where path-to-apk-file is the file system path to your app's APK. 
    adb install path-to-apk-file
  3. If your installation was successful, adb will respond with a message similar to this one: 
     764 KB/s (217246 bytes in 0.277s)
 pkg: /data/local/tmp/HelloWorld.apk
 Success
  4. To re-install an app that already exists on the device, you can use the -r option to reinstall it. 
     adb install -r path-to-apk-file

Running your Brightcove {{ site.product_short }} App

Sideloaded apps appear in both the Recent row and in the My Library row in the Apps section. You can also find your app in the Settings menu.

  1. From the Amazon Fire TV main screen, select Settings > Applications > Manage Installed Applications.
  2. Select the Brightcove {{ site.product_short }} app.
  3. Select Launch application.

Installing the Fire TV app with Amazon Live App Testing service

In this section, you will learn how to install and test the Fire TV app with the Amazon Live App Testing service.

Use this installation procedure in case you want to test the App internal payment methods, such as subscriptions or event payments.

To be able to install the Fire TV app in your device, you will need a Live App Testing email invitation from Brightcove.

In the case you are not receiving the invitation email for the Live App Testing, please follow this link and enable marketing communication for Appstore by selecting the option labeled Send me marketing email from the following categories and click Save.

After setting the marketing communication preferences, the change will start with the next and all subsequent email invitation tests.

Once you receive the email invitation to participate in the Live App Testing, please perform the following steps:

  1. In your email invitation, click the link for the marketplace that you will be testing

    The link will open the retail page for the Brightcove {{ site.product_short }} App in the specified marketplace.

  2. On the app retail page, from the Deliver To: drop-down menu, select the device where you want to install the Brightcove {{ site.product_short }} app.
  3. If a particular device is not available in the tester's current marketplace, that device will not appear in the drop-down menu. If none of the devices that your app targets are available in a given marketplace, the drop-down menu will be replaced with a Continue button on the app's retail page.
  4. Install the Brightcove {{ site.product_short }} app to the specified device. You can test on as many devices as you have associated with your Amazon account that the app targets.
  5. From the Amazon Fire TV main screen, select Settings > Applications > Manage Installed Applications.
  6. Select the Brightcove {{ site.product_short }} app.
  7. Select Launch application.

Uninstalling your Brightcove {{ site.product_short }} App

To uninstall your app from the Fire TV device follow these steps:

  1. From Fire TV's main screen, select Settings > Applications > Manage Installed Applications.
  2. Select your Brightcove {{ site.product_short }} app.
  3. Select Uninstall > Uninstall
